CSS Selector

Applying CSS to Different parts of HTML

  1. The element selector target element based on their HTML tag name
  2. Class selector target element based on the value of the clss attribute
  3. ID selector target element based on the value of ID attribute
  4. Attribute selector target the element based on the attribute and the value
  5. The universal elector target all the element
